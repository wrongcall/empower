Advanced Power Calculator is a Python-based command-line calculator that performs various mathematical operations beyond basic arithmetic.

It includes:

Power Calculations (x^y)
Square Root
Logarithm (Base 10)
Percentage Calculator
Compound Interest Calculator
Calculation History
Error Handling
Interactive Menu System



1. Power Calculation

Calculate powers and exponents.

Example:

2 ^ 5 = 32
2. Square Root

Example:

√81 = 9
3. Logarithm

Example:

log10(1000) = 3
4. Percentage Calculator

Example:

50 out of 200 = 25%
5. Compound Interest

Calculate investment growth using compound interest.

Formula:

A = P(1 + r/n)^(nt)
6. History Tracking

Stores previous calculations during runtime.

7. Error Handling

Prevents crashes from:

Invalid inputs
Division by zero
Negative square roots
Invalid logarithm values
